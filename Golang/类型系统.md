#### 组合类型

1.  指针类型
2.  数组，切片，映射类型
3.  函数类型
4.  通道类型
5.  结构体类型
6.  接口类型



*   定义类型，在某个类型类型声明定义中定义的类型，所有基本类型都是定义类型
*   非定义类型，一个非定义类型一定是一个组合类型

```go
type T1 []string // T1是定义类型，[]string本身是非定义类型
type T2 = T1 // T2是T1的类型别名，也是定义类型
type T3 = []string // T3是[]string的类型别名，非定义类型
```
